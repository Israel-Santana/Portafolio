Acerca de Mí

Yo soy un Frontend Developer

Con una pasión innata por el diseño y la tecnología, me especializo en crear experiencias web intuitivas y atractivas. A lo largo de mi carrera, he desarrollado habilidades sólidas en HTML, CSS y JavaScript, así como en frameworks modernos como React y Angular. Mi objetivo es siempre mejorar la usabilidad y accesibilidad de las aplicaciones web, garantizando que cada proyecto no solo sea visualmente impactante, sino también funcional y fácil de usar. Me encanta colaborar con equipos creativos y enfrentar desafíos que me permitan crecer profesionalmente y aprender nuevas tecnologías.

Mi Portafolio

En esta sección encontrarás una selección de mis trabajos más destacados como desarrollador frontend. Cada proyecto refleja mi enfoque en la creación de experiencias web intuitivas, responsivas y funcionales. Desde aplicaciones interactivas hasta herramientas útiles, estos proyectos muestran mi habilidad para resolver problemas complejos y ofrecer soluciones innovadoras.

"Adivina el Número Secreto"

Es un juego de lógica y suerte creado con HTML, CSS y JavaScript. El objetivo es adivinar un número secreto generado aleatoriamente.

"Encriptador de Texto"

Es una aplicación web desarrollada utilizando HTML, CSS y JavaScript que permite a los usuarios encriptar y desencriptar mensajes de forma sencilla.

"Alura Play"

Esta desarrollado con HTML y CSS. Ofrece una interfaz moderna y responsiva, destaca por su simplicidad y eficiencia en el diseño visual, recreando la experiencia básica de navegación y visualización de videos.
